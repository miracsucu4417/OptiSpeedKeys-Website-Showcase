# OptiSpeedKeys - On Parmak Hız Testi

Merhaba! Bu repository, benim tarafımdan geliştirilen **OptiSpeedKeys - On Parmak Hız Testi** web sitesini sergilemek amacıyla oluşturulmuştur.
